
# 环境设置

为了设置您的环境以在此处运行代码，首先安装所有要求：
```shell
pip install -r requirements.txt
```
### openai_api_key设置
将模板example.env复制到.env,并在文件.env中适当地编辑变量。
```
OPENAI_API_KEY:你自己的openaikey
http_proxy/s：你自己的代理端口
```
